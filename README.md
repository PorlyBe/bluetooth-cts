This repository contains the Bluetooth CTS (Current Time Service) add-on for Home Assistant.

## How to use

[![Open your Home Assistant instance and show the add add-on repository dialog with a specific repository URL pre-filled.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2FPorlyBe%2Fbluetooth-cts)


1. In Home Assistant, go to **Settings > Add-ons > Add-on Store**.
2. Click the three dots in the top right and select **Repositories**.
3. Add this repository URL: `https://github.com/PorlyBe/bluetooth-cts`
4. The Bluetooth CTS add-on will appear in the store for installation.
